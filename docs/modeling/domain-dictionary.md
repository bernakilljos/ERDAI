# 도메인 사전

테이블명 prefix 기반 도메인 분류 기준.

| 도메인 | 설명 | 예시 |
|---|---|---|
| AUTH | 인증/권한 | auth_user, auth_role |
| PRJ | 프로젝트 | prj_project |
| CFG | 설정 | cfg_db_connection |
| MD  | 메타데이터 | md_snapshot, md_table |
| ERD | ERD 결과 | erd_inferred_relation |
| SYS | 공통 코드 | sys_code, sys_code_group |
| ETC | 기타 | 기타 테이블 |
